# BetterAnimations


BetterAnimations could change your hole discord look join today!!


# Requirments

install git
Windows - https://git-scm.com/install/windows
MacOS - https://git-scm.com/install/mac
Linux - https://git-scm.com/install/linux


# Second do these steps

first run

git clone https://github.com/Vendicated/Vencord.git

then 

cd Vencord

then

pnpm install

then

pnpm build

then

pnpm inject

it will ask you to install at your Discord Directory Click Enter

then restart your discord

# MUST HAVE DISCORD INSTALLED

now open Settings
then Click Plugins
search for BetterAnimations and Enable it

# YOU INSTALLED BETTER ANIMATIONS
